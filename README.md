# Complex Number Operations in C++

This is a simple Object-Oriented Programming (OOP) project in C++ that performs basic arithmetic operations on complex numbers.

## Overview

The program defines a `ComplexNumber` class and supports operations like:

- Addition
- Subtraction
- Multiplication
- Input and Output formatting

## Features

- Class-based implementation
- Constructor overloading
- Complex number arithmetic
- Operator-style functions (add, sub, mul)
- Formatted output display

## OOP Concepts Used

- Class and Object
- Encapsulation
- Constructors
- Member Functions
- Const correctness
